## 📌 Project Overview

The **Loan Risk & Default Analysis Dashboard** is an end-to-end **Data Analytics** project designed to analyze loan applications, borrower demographics, financial behavior, and default risk.

The project follows a complete analytics workflow starting from **Microsoft Excel** for raw data handling, **SQL Server** for data storage, **Power Query** for transformation, **DAX** for business calculations, and **Power BI** for creating an interactive dashboard.

The dashboard enables stakeholders to monitor lending performance, understand borrower characteristics, identify financial risk patterns, and support informed business decisions through data visualization.
