

# Data Dictionary

This document describes the key fields used in the Loan Risk & Default Analysis Dashboard.

| Column Name | Description |
|-------------|-------------|
| Loan ID | Unique identifier for each loan |
| Age | Applicant's age |
| Gender | Applicant's gender |
| Marital Status | Marital status of the applicant |
| Education | Education level |
| Employment Type | Employment category |
| Income | Annual income of the applicant |
| Credit Score | Applicant credit score |
| Loan Amount | Approved loan amount |
| Loan Purpose | Purpose of the loan |
| Interest Rate | Interest rate applied |
| Loan Term | Loan repayment duration |
| Default Status | Indicates whether the loan defaulted |
