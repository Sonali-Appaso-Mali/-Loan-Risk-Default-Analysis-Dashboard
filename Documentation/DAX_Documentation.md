
# DAX Documentation

This document explains the DAX measures used in the Power BI dashboard.

## Sample Measures

### Total Loan Amount
Calculates the total approved loan amount.

### Average Income
Calculates the average applicant income.

### Default Rate (%)
Calculates the percentage of defaulted loans.

### Year-over-Year Loan Growth
Measures annual change in loan amount.

### Year-over-Year Default Growth
Measures annual change in default loans.

> Additional DAX measures can be added as the dashboard evolves.
